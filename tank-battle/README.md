# 网页版《坦克大战》（单人 vs AI）

## 操作
- 方向键：移动 / 转向
- 空格：射击（有射速限制）
- P：暂停 / 继续
- R：重新开始
- Enter：过关后进入下一关

## 规则
- 地图边界为**钢墙**（不可摧毁）
- **砖墙**可被子弹摧毁
- 清空所有敌方坦克即可进入下一关
- 玩家共有 3 条生命，被击中会掉 1 条并在出生点复活

## 本地运行
这是纯静态前端项目（无需构建）。

任选其一：
1. 直接双击 `index.html`（部分浏览器安全策略下可能影响键盘事件，一般仍可运行）
2. 推荐：在本目录启动一个静态服务器，例如：
   - Python：`python -m http.server 8000`
   - Node：`npx serve`
   然后浏览器打开 `http://localhost:8000/`
